#  WEEK 3 REPORT  

##  TITLE  
MediClaimAISystem  

---

##  WORK DONE  
During this week, our team shifted from research and analysis to the development phase of the project. The primary focus was on designing and building the core backend system for the MediClaimAISystem.

We started by setting up the backend using FastAPI, which allowed us to build a fast and efficient API structure. The project architecture was organized into multiple modules such as routes, services, models, and utilities to maintain a clean and scalable design.

A major achievement this week was the initial development of the AI-based document processing model. The system was designed to accept medical documents and begin extracting relevant information required for claim processing.

Additionally, we implemented a secure authentication system using OTP (One-Time Password). This included:
- OTP verification for users  
- Admin login authentication  
- Secure access control for system usage  

We also worked on integrating basic API endpoints to handle:
- File uploads  
- User authentication  
- Initial data processing
